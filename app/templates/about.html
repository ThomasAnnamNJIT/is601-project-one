Docker

Docker is a containerization technology tool for creating and managing containers. The core building blocks for Docker include containers and images. Images are the blueprints for building a container which are commonly defined via a Dockerfile or a docker-compose.yml. Within a Dockerfile, developers providers instructions to build a container, this is done via commands such as FROM, RUN, CMD, ENTRYPOINT, etc. Each Dockerfile must be built from a base image, which must be the first instruction within a Dockerfile (i.e FROM). Even though you can pull base images from a custom registry such as ECR, most base images come from DockerHub, which is essentially the app store/repository for all base images). Some of the Docker commands I know include:

  • docker build -t .
  • docker kill "container id"
  • docker system prune
  • docker run image id
  • docker compose up
  • docker compose down
